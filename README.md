# Nixie

Local system dashboard served on http://localhost:8765

## Requirements
- Windows
- Python 3.x

## Setup
```powershell
py -m venv .venv
.\.venv\Scripts\python.exe -m pip install --upgrade pip
.\.venv\Scripts\pip.exe install -r requirements.txt
```

## Run
```powershell
.\.venv\Scripts\python.exe .\main.py
```

Open: http://localhost:8765
Stream: http://localhost:8765/stream
